# Core-Novaness Email Templates

These templates are based on the current Core-Novaness site tone and brand colors.

## Files

- `reply-template.html` for email providers that support saved HTML templates or rich-text canned responses
- `reply-template.txt` for providers that work better with plain text

## How to use

1. Open the file you want to use.
2. Replace each bracketed placeholder such as `[Client First Name]` and `[Your Name]`.
3. Save the edited version in your email provider as a reusable reply template.

## Notes

- The HTML version avoids SVG and web fonts so it holds up better in Gmail and Outlook.
- If your provider strips styles, use the plain-text version instead.
